Chzzk Chatfilter (치지직 채팅 필터)
- 네이버 스트리밍 플랫폼 '치지직(Chzzk)'의 채팅창을 쾌적하게 관리하고, 편의 기능을 제공하는 크롬 확장 프로그램입니다.
- 

📌 주요 기능
- 채팅 필터링: 특정 키워드, 사용자, 이모티콘을 차단합니다.
- 간편 차단 단축키: 채팅창 내에서 Alt + 좌클릭으로 즉시 차단할 수 있습니다.
- 

UI 요소 제어:
- 고정 메시지 차단
- 후원 메시지 차단
- 주간 후원 랭킹 차단
- 진행 중인 미션 차단


차단 방식 선택:
- 완전 삭제 (메시지 자체를 없앰)
- 텍스트만 삭제 (닉네임은 유지)
- 차단 알림 표시 ("차단된 메시지입니다"로 대체)
- 자동 수집: 포인트(통나무) 버튼 자동 클릭
- 설정 백업: 차단 목록 내보내기/불러오기 지원


🚀 설치 방법
이 확장 프로그램은 현재 크롬 웹 스토어에 등록되지 않았으며, 수동으로 설치해야 합니다.

1. 이 저장소를 Clone 하거나 ZIP 파일로 다운로드하여 압축을 풉니다.
2. 크롬 브라우저 주소창에 chrome://extensions를 입력하여 이동합니다.
3. 우측 상단의 **개발자 모드(Developer mode)**를 켭니다.
4. 좌측 상단의 압축 해제된 확장 프로그램을 로드합니다(Load unpacked) 버튼을 클릭합니다.
5. 다운로드 받은 폴더(manifest.json이 있는 폴더)를 선택합니다.


📖 사용 가이드

1. 간편 차단 (단축키)
방송 시청 중 채팅창에서 다음과 같이 조작하여 즉시 차단 목록에 추가할 수 있습니다.
- 사용자 차단: 닉네임 위에 마우스를 올리고 Alt + 좌클릭
- 이모티콘 차단: 이모티콘 위에 마우스를 올리고 Alt + 좌클릭


2. 필터 설정 (팝업 메뉴)
브라우저 우측 상단의 확장 프로그램 아이콘을 클릭하여 설정을 변경할 수 있습니다.

설정 탭:
- 각종 필터(채팅, 후원, 랭킹, 미션)의 켜짐/꺼짐 상태를 토글합니다.
- 차단된 메시지를 처리하는 방식(삭제/텍스트 삭제/알림)을 선택합니다.

차단 목록 탭:
- 등록된 키워드 및 이모티콘 리스트를 확인하고 삭제할 수 있습니다.
- 수동으로 키워드를 입력하여 추가할 수 있습니다.
- 설정 데이터를 JSON 파일로 백업하거나 복원할 수 있습니다.


3. 포인트 자동 수집
- 확장 프로그램이 활성화되어 있다면, 방송 시청 중 발생하는 포인트 상자를 별도 조작 없이 자동으로 수령합니다.


⚠️ 주의사항
- 네이버 치지직의 웹사이트 구조가 변경될 경우 기능이 정상적으로 작동하지 않을 수 있습니다.
- 이 프로그램은 개인 사용 목적으로 개발되었습니다.
