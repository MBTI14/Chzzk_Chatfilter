# 치지직 클린 필터 & 자동 수령 (Chzzk Chat Filter)

치지직 채팅창에서 보기 싫은 단어, 이모티콘을 차단하고 포인트(통나무)를 자동으로 수령해주는 크롬 확장 프로그램입니다.

## ✨ 주요 기능
- **단어 차단:** 설정한 단어가 포함된 채팅을 숨깁니다.
- **이모티콘 차단:** 채팅창 이미지 우클릭으로 이모티콘을 쉽게 차단합니다. (다시보기에선 우클릭 메뉴가 작동하지 않아요.)
- **사용자 차단:** 특정 사용자의 채팅을 차단합니다.
- **포인트 자동 수령:** 방송을 보는 동안 포인트를 자동으로 클릭해줍니다.

## 📥 설치 방법
1. 오른쪽 **Releases** 메뉴에서 최신 버전 `Chzzk_Chatfilter v1.0`의 **ZIP 파일**을 다운로드하세요.
2. 다운로드한 파일의 압축을 원하는 곳에 풉니다.
3. 크롬(또는 엣지) 주소창에 `chrome://extensions`를 입력해 이동합니다.
4. 우측 상단 **'개발자 모드'**를 켭니다.
5. **'압축해제된 확장 프로그램을 로드합니다'** 버튼을 누르고 압축 푼 폴더를 선택하세요.

## ⚠️ 주의사항
- 채팅창 구조가 변경되면 기능이 작동하지 않을 수 있습니다.
- 개인적인 용도로 제작되었으며 오류가 있을 수 있습니다.
