**설치 방법**
1. 우측 `Releases`를 클릭하세요.
2. `Chzzk.Chatfilter.v1.0.zip` 을 다운로드 하고 아무데나 압축을 푸세요.
4. 크롬 브라우저 주소창에 `chrome://extensions/` 을 입력하여 `확장 프로그램 관리` 창으로 이동하세요.
5. 우측 상단의 `개발자 모드`를 켜세요.
6. 좌측 상단의 `압축해제된 확장 프로그램 로드` 를 클릭하세요.
7. 다운로드 받은 폴더 `Chzzk.Chatfilter.v1.0` 자체를 선택하세요.

**사용 방법**

- `설정` 탭의 채팅창의 `고정, 후원, 주간 후원 랭킹, 진행 중인 미션` 을 차단할 수 있어요.

- `차단 목록` 에서 차단했던 아이디, 이모티콘, 키워드를 해제할 수 있어요.
